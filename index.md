You can create both ordered and unordered lists:

Unordered List:

- Item 1

- Item 3

Ordered List:

1. First Item
2. Second Item
3. Third Item

## Code

You can also highlight code using backticks:

Inline code: `print("Hello, world!")`

Code block:

```python
def greet(name):
    print(f"Hello, {name}!")
greet("Nanda")
```
